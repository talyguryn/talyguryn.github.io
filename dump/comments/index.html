Horray! Now you can add comments to any page of your site with a few lines of code.

No database, no server, no nothing. Just a simple HTML file and a simple JavaScript file.

All page comments will be stored as comments for an issue in the target repository.

Thanks to https://utteranc.es/ for the awesome comments section.

Just added the following code to the page:

<script src="https://utteranc.es/client.js"         repo="talyguryn/talyguryn.github.io"         issue-term="pathname"         label="comments"         theme="github-light"         crossorigin="anonymous"         async> </script>

You can place it wherever you want to add comments block.
