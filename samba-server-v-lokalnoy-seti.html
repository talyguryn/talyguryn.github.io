Samba сервер в локальной сети

Виталя March 4, 2025

Устанавливаем сервис.

apt update
apt install -y samba

Открываем настройки.

nano /etc/samba/smb.conf

Добавляем настройки папки, которой хотим поделиться.

[Transmission]
  path = /root/storage
  browseable = yes
  read only = no
  guest ok = yes
  create mask = 0777
  directory mask = 0777
  force user = root

Перезагружаем сервис samba.

systemctl restart smbd

Теперь эта папка будет доступна в локальной сети с других устройств.
